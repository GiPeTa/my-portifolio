1# Sobre-Mim

Olá, meu nome é Giovanni Pereira sou desenvolvedor desde 2019. Meu primeiro contato com a programação foi através do desenvolvimento de Bots no Discord. Esse primeiro contato me encantou e me fez querer aprender JavaScript, depois disso foi implementando cada vez mais tecnologias em minha aplicações assim entregando as melhores soluções para meus clientes, implementando primeiramente um banco de dados local e depois migrando para um online.

Atualmente estou focado em Desenvolvimento Web, atuando como programador e designer, alinhando assim os conhecimento das duas áreas conseguindo entregar o melhor projeto possível, com HTML, CSS e JavaScript, e grandes diferenciais como responsividade, designing de logos e páginas, utilizando Psicologia das Cores para entregar a paleta de cores mais adequada para o projeto.

4# Projetos

Bot Discord v14

Esse bot de moderação do Discord que eu criei é uma ferramenta eficiente e personalizável para manter a ordem e a segurança em servidores do Discord. Ele executará tarefas de moderação automaticamente, monitora atividades suspeitas e pode ser configurado para atender às necessidades específicas de cada servidor. O bot é fácil entendimento e ajuda a economizar tempo e esforço dos moderadores.

JavaScript Node.js Npm MongoDb

Site CBC

Site desenvolvido para o Campinas Bike Clube uma organização social apresenta uma interface limpa e moderna, construída usando HTML e CSS. A navegação é intuitiva e permite que os visitantes encontrem facilmente informações importantes sobre a organização e suas iniciativas. O site é altamente responsivo e se adapta a qualquer dispositivo, tornando a experiência do usuário agradável em qualquer plataforma.
